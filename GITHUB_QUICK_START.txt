================================================================================
                            ⚡ 2-MINUTE SETUP
================================================================================

STEP 1: Create GitHub Personal Access Token
  1. Go to https://github.com/settings/tokens
  2. Click "Generate new token (classic)"
  3. Give it a name: "ecosystem-token-deploy"
  4. Check these permissions:
     ☑ repo (full control of private repositories)
     ☑ workflow (Actions)
  5. Click "Generate token"
  6. COPY THE TOKEN (you'll use it next)

STEP 2: Create GitHub Repository
  1. Go to https://github.com/new
  2. Repository name: ecosystem-token
  3. Description: "Solana collateral-backed ecosystem token"
  4. Choose: Public or Private
  5. Click "Create repository"

STEP 3: Push Your Code
  Option A (Automated):
    bash /home/claude/push-to-github.sh
    → Follow the prompts
    → Paste your GitHub token when asked
    → Done!

  Option B (Manual):
    cd /home/claude
    git init
    git config user.name "Your Name"
    git config user.email "your@email.com"
    git add .
    git commit -m "Initial commit: Complete ecosystem token"
    git branch -M main
    git remote add origin https://YOUR_USERNAME:YOUR_TOKEN@github.com/YOUR_USERNAME/ecosystem-token.git
    git push -u origin main

STEP 4: Watch Build
  1. Go to https://github.com/YOUR_USERNAME/ecosystem-token
  2. Click "Actions" tab
  3. Watch the workflow run automatically
  4. Build takes ~10 minutes

STEP 5: Get Program ID
  1. When build completes, click the workflow
  2. Find log line: "Program deployed: ABC123..."
  3. That's your PROGRAM_ID
  4. Save it for dashboard

================================================================================
                           🔑 OPTIONAL: AUTO-DEPLOY
================================================================================

To make GitHub automatically deploy to devnet on every push:

1. Generate Solana keypair for CI/CD:
   solana-keygen new -o ci-keypair.json --no-bip39-passphrase

2. Go to your GitHub repo → Settings → Secrets and variables → Actions

3. Click "New repository secret"
   Name: SOLANA_KEYPAIR
   Value: (paste entire contents of ci-keypair.json)

4. Click "Add secret"

Now every push will automatically deploy to devnet!

⚠️  Important: Use a SEPARATE keypair for GitHub Actions (not your main wallet)
